User Guide for the Copy Simulation Bot

This guide explains how to use the copy simulation bot via Telegram. The bot allows you to:

  • Load wallet activities.
  • Configure parameters such as fee, purchase amount, tip, etc.
  • Simulate operations either by fully replicating wallet trades or by applying your custom strategy.
  • Receive a final simulation report with detailed information for each token.

1. Choosing the Language

When you start the bot with the /start command, you will be prompted to choose a language:

  • Tap English for English.
  • Tap –†—É—Å—Å–∫–∏–π for Russian.

After selecting your language, the bot will send a welcome message and display the main menu.

2. Main Menu

The main menu provides the following buttons (each on a separate row):

  • ‚öôÔ∏è Settings ‚Äì to configure fees, purchase amount, tip, etc.
  • üß© Strategy ‚Äì to select and customize the simulation strategy.
  • üìä Copy Wallet Simulation ‚Äì to load wallet activities and start simulation.

3. Configuring Parameters

Tap the Settings button to set your parameters. In the settings interface you can change:

  • Fee (in SOL): the fee charged per operation.
  • Tip (in SOL): an additional charge per operation.
  • Trading Bot Commission (in %): the commission applied on purchases.
  • Purchase Amount (in SOL): the amount used for simulating each purchase.

After updating, the bot will display your current settings.

4. Strategy Selection

Tap the üß© Strategy button from the main menu to choose your simulation mode:

  • üîÑ Full replication of deals ‚Äì the bot will simulate the wallet‚Äôs operations exactly as they were executed. In this mode, there are two variants available:
    • 1) Replicate ALL trades ‚Äì the simulation will replicate every operation of the wallet.
    • 2) First buy + all sells ‚Äì the bot will use the price from the first buy to calculate all subsequent sell operations.
  • üõ† Custom strategy ‚Äì you can create, edit, and delete individual orders (e.g. Take Profit and Stop Loss orders). In custom strategy mode:
    • Tapping Create order lets you choose an order type and input its parameters.
    • Your orders are listed in the ‚ÄúYour strategy:‚Äù message, where each order has buttons for editing or deletion.
    • A Back button returns you to the strategy selection menu.

5. Loading Wallet Activities

  1. Tap Copy Wallet Simulation.
  2. Enter the wallet address.
  3. Select the period for loading activities (e.g., 1 day, 1 week, 1 month, or specify a custom period).
  4. The bot loads activities using pagination (100 records per request) and filters them by the chosen period.
  5. Once loaded, you will see a message like: ‚úÖ Received 25 activities for the selected period.

6. Simulating Operations

After loading the wallet activities, tap the Start Simulation button. The bot will:

  • Display a message indicating that the simulation has started, along with a progress indicator.
  • Simulate the wallet‚Äôs operations (both buys and sells) in the same order and proportions, or apply your custom strategy if selected.
  • Calculate the purchase or sale price for each operation using historical data.
  • Use your configured settings (purchase amount, fee, tip, and commission) in the simulation.
  • Compute metrics for each token, including ROI, PnL, and the number of buy and sell operations.

7. Receiving the Report

After the simulation, the bot sends a final message with buttons:

  • üìÑ Get Detailed Report ‚Äì tapping this button will send you an Excel file containing the full simulation report with detailed information for each token.
  • üè† Main Menu ‚Äì to return to the main menu.
